Multiple Linear Regression

Multiple Linear Regression is also similar to Simple Linear Regression model. But there is a small difference which plays a mojor role in "Multiple Linear Regression", the value of a dependent variable is predicted based on more than one independent variables. Where as in "Simple Linear Regression", only one independent variable.

Multiple Linear Regression equation -

y=b+m1.x1+m2.x2+.....+mn.xn

where,
y -> dependent variable
x1,x2,....,xn -> independent variable
b -> intercept
m1,m2,....,mn -> slope

By finding the "Mean Squared Error" we are trying to minimize the errors in prediction.
"Line of best fit" - the regression line from the errors would be minimal. We are trying to minimize the length between the observed value (yi) & the predicted value from our model (yp).
